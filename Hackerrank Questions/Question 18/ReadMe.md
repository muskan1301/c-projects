# Question 18
In this example, you will learn to check whether a number entered by the user is even or odd Using the Ternary Operator.

### Input Format
```
A Number.
```
### Constraints

Entered number will be an integer

### Output Format
```
Whether entered number is odd or even
```
### Sample Input 0
```
33
```
### Sample Output 0
```
33 is odd.
```
