# Question 22
Given an integer N as input, find whether it is even or not using bitwise operator.

### Input Format
```
50
```
__where:__

First line represents the integer N.

### Constraints

Value of N can be in the range 0 to 10000.

### Output Format
```
Print whether number is even or odd
```
### Sample Input 0
```
55
```
### Sample Output 0
```
Odd
```
